== Description ==

Plugin para mostrar el precio de los productos a 3, 6, 9, 12 Meses sin Intereses en la pagina individual de productos de WooCommerce, antes del boton de añadir al carrito. Hecho para tiendas de comercio electronico en el Buen Fin en México.

== Installation ==

Como instalar el plugin.

1. Descargar el archivo `buen-fin-woo.zip` y copiarlo dentro de la carpeta `/wp-content/plugins/`.
2. Activar el plugin en el menu de 'Plugins' de WordPress.
3. Dirigete a la configuracion de WooCommerce > Productos > Buen Fin, aqui podras seleccionar cuales son lo meses que quieres mostrar.

== Frequently Asked Questions ==
